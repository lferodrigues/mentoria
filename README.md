- Responsividade

A plataforma apresenta um curso completo de informática voltado para iniciantes e profissionais que desejam desenvolver habilidades digitais essenciais para o mercado de trabalho.

O conteúdo aborda:

- Microsoft Excel
- Microsoft Word
- PowerPoint

- Inteligência Artificial
- Canva
- Segurança da Informação
- LGPD
- Manutenção de Computadores
